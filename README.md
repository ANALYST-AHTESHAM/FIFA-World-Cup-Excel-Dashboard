# 🏆 FIFA World Cup Excel Dashboard

An interactive, visually rich **Excel dashboard** that summarizes the complete history and results of the **FIFA World Cup** — built entirely with native Excel features (PivotTables, PivotCharts, slicers, and dynamic formulas). No add-ins, no macros — just clean data modeling and design.

![FIFA](((https://github.com/ANALYST-AHTESHAM/FIFA-World-Cup-Excel-Dashboard/blob/main/FIFA%20World%20Cup%20Excel%20DashBoard%20-%20Excel%2008-07-2026%2019_17_21.png)assets/dashboard-preview.png))
## 📊 Overview

This dashboard delivers a one-page, executive-style summary of FIFA World Cup history — perfect for football fans, data analysts, or anyone looking to showcase Excel dashboarding skills in a portfolio.

### Key Highlights Displayed
- 🏟️ **Host Country & Final Standings** — Champion, Runner-up, Bronze Medal, and 4th Place
- ⚽ **Top 5 All-Time Goal Scorers**
- 📈 **Total & Average Attendance** across tournaments
- 🌍 **Host Countries Map (Treemap)** by year
- 🏆 **Champions by Country** — total attendance per winning nation
- 🟨 **Yellow & Red Card Statistics**
- 🎖️ **Award Winners** — Golden Ball, Golden Boot, Golden Gloves, Best Young Player
- 🗓️ **Tournament Timeline (1930–2022)**
- 🇺🇳 **Full List of Participating Countries**

---

## 🛠️ Tools & Techniques Used

| Feature | Purpose |
|---|---|
| PivotTables & PivotCharts | Dynamic summarization of tournament data |
| Slicers | Interactive filtering by year/country |
| Treemap Chart | Visualizing host countries by frequency |
| Conditional Formatting | Highlighting key stats |
| Card-style Layout | KPI cards for Champion, Runner-up, etc. |
| Custom Theme & Icons | Professional, branded dashboard look |

## 🚀 How to Use

1. Download or clone this repository.
2. Open `FIFA_World_Cup_Dashboard.xlsx` in **Microsoft Excel (2016 or later recommended)**.
3. Use the slicers/filters to explore data by year, country, or stage.
4. All charts and KPI cards update automatically based on selections.
